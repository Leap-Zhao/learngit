## 0.设置相关

  - 显示行号 :set nu 或 :set number
  - 不显示行号 :set nonu 或 :set nonumber
  - u是撤销你刚才做的动作
  - ctrl+r 是恢复你刚才撤销的动作


## 1.复制,删除,剪切

### 1-1 删除
 - 删除当前行 dd
 - 删除5-10行  :5,10d
 - 删除光标位置到本行结尾 d$
 - 删除光标位置到本行开头 d0
 
### 1-2 剪切
  - 剪切5-10行到20行 :5,10m20

### 1-3 复制
  - 单行复制 在命令模式下，将光标移动到将要复制的行处，按“yy”进行复制；
  - 多行复制 在命令模式下，将光标移动到将要复制的首行处，按“nyy”复制n行；其中n为1、2、3⋯⋯
  - 粘贴4遍  4p
  
  
## 2.跳转

### 跳转字符
  - h,j,k,l 为 左 下 上 右,前面加数字n表示一次动的数
  - 向下5行  5j

### 跳转到指定行
  - 跳转到文件开头  gg  或 :1
  - 跳转到文件尾  G  或 :$
  - 跳转到指定行(比如第54行)  54G 或 :54
  
### 翻页
  - 向下翻半页  ctrl + d
  - 向上翻半页  ctrl + u
  - 向上翻一页  ctrl + b
  - 向下翻一页  ctrl + f

### 其它 
  - Ctrl+o 来回到之前修改的地方
  - Ctrl+i 会回退上面的跳动。
